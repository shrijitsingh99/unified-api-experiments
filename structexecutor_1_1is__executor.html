Detailed Description

template<class T, typename = void>
struct executor::is_executor< T, typename >

Checks whether the type is an executor.

A given type T is an Executor if it satisfies the following properties:

  1. Provides a function named execute that eagerly submits work on a single execution agent created for it by the executor
  2. Is a CopyConstructible type
  3. Is a EqualityComparable type
