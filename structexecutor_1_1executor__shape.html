Detailed Description

template<typename Executor, typename = void>
struct executor::executor_shape< Executor, typename >

A given Executor can have a custom shape for bulk execute.

By default if not explicitly specified by the executor the shape is std::size_t. The shape represents the number of execution units and their dimensionality if needed.

executor_shape is an Executor type trait which provides the shape type defined by the Executor.

Any executor can define a custom shape by defining the alias shape_type for the custom shape.
