Detailed Description

template<typename DerivedProperty, bool requireable, bool preferable>
template<typename T>
struct executor::base_executor_property< DerivedProperty, requireable, preferable >::is_applicable_property< T >

Checks whether the given Property is applicable for the given type T.

Currently for the property to be applicable T should be an executor type. In proposal P1393R0 there are no conditions specified for a property to be applicable ans is left to the implementer to decide.

Part of Proposal P1393R0
